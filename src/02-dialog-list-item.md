---
title: Dialog List Item
subtitle: Dan's Little Books style dialog
---

To use **Dialog List Item** extensions,
name a item by `dialog`, like the following:

``` markdown
- [dialog]

  - What is number?

  - Number is about counting.

    1, 2, 3, ...
```
