section: Kinds of Renders
title: Book
---

We render `Book` using [readonly.link/books](https://readonly.link/books).

Example `book.json`:

``` json
{
  "title": "The Little Typer Exercises",
  "subtitle": "A student's report on \"The Little Typer\"",
  "version": "0.0.2",
  "authors": [
    "Xie Yuheng"
  ],
  "date": "2021",
  "src": "src"
}
```
